1. Model Warna

1.1 RGB (Red, Green, Blue)

Penggunaan: Monitor, TV, layar digital (additive color model)

Model warna yang paling umum di komputer grafis. Setiap warna adalah kombinasi dari Red, Green, dan Blue.

R: 0-255 (Red)
G: 0-255 (Green)
B: 0-255 (Blue)
A: 0-1 (Alpha/Transparansi)

1.2 HSV (Hue, Saturation, Value)

Penggunaan: Color picking, image editing (lebih intuitif untuk manusia)
H (Hue): 0-360¬∞ (warna)
S (Saturation): 0-100% (intensitas warna)
V (Value): 0-100% (kecerahan)

1.3 CMYK (Cyan, Magenta, Yellow, Black)

Penggunaan: Printing (subtractive color model)
Model Komponen Penggunaan Jenis
RGB Red, Green, Blue Monitor, layar digital Additive
HSV Hue, Saturation, Value Color picker, editing -
CMYK Cyan, Magenta, Yellow, Black Printing Subtractive

2. Color Blending

Konsep: Menggabungkan dua warna untuk menghasilkan warna baru

2.1 Alpha Blending

Teknik paling umum untuk transparansi:

Result = (Source √ó Alpha) + (Destination √ó (1 - Alpha))

2.2 Blend Modes

Mode Rumus Hasil
Normal src √ó alpha + dst √ó (1-alpha) Standar transparansi
Multiply src √ó dst Gelap (seperti menumpuk cat)
Screen 1 - (1-src) √ó (1-dst) Terang (kebalikan multiply)
Overlay Kombinasi multiply & screen Kontras tinggi

3. Dasar Pencahayaan

Konsep: Simulasi cahaya untuk memberi kesan 3D pada objek 2D

3.1 Komponen Cahaya

  1. Ambient Light: Cahaya merata dari semua arah
  2. Diffuse Light: Cahaya yang menyebar saat mengenai permukaan
  3. Specular Light: Pantulan cahaya (highlight)
Total Light = Ambient + Diffuse + Specular
Final Color = Object Color √ó Total Light

4. Transformasi 2D

Konsep: Mengubah posisi, ukuran, atau orientasi objek

4.1 Translasi (Translation)

Menggeser objek ke posisi baru:

x' = x + dx
y' = y + dy

4.2 Rotasi (Rotation)

Memutar objek terhadap titik pusat:

x' = x √ó cos(Œ∏) - y √ó sin(Œ∏)
y' = x √ó sin(Œ∏) + y √ó cos(Œ∏)

4.3 Skala (Scaling)

Memperbesar atau memperkecil objek:

x' = x √ó sx
y' = y √ó sy
