# Grafika Komputer - Praktikum

## 🚀 Cara Mulai (3 Langkah)

```bash
npm install
npm run build
npm run serve
```

Buka browser: **http://localhost:8080/index-baru.html**

1. **Piksel vs Vektor** - Perbedaan representasi gambar

5. **Blending** - Pencampuran warna
6. **Pencahayaan** - Dasar lighting
