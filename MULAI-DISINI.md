## 🆘 Troubleshooting

### Server tidak jalan?
```bash
npm run serve
# atau
npx http-server . -p 8080
```

### TypeScript error?
```bash
rm -rf dist/
npm run build
```

### Canvas tidak muncul?
- Buka console browser (F12)
- Pastikan tidak ada error JavaScript
- Pastikan sudah `npm run build`
