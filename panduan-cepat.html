üìñ Konsep Dasar

1. Gambar Piksel (Raster/Bitmap)

Gambar tersusun dari grid piksel-piksel kecil:

  • Setiap piksel menyimpan warna RGBA (Red, Green, Blue, Alpha)
  • Resolusi tetap (misalnya 800√ó600 piksel)
  • Kehilangan kualitas saat diperbesar (pixelated)
  • Contoh format: PNG, JPG, BMP

2. Gambar Vektor

Gambar menggunakan rumus matematis:

  • Disimpan sebagai shapes (lingkaran, garis, polygon)
  • Resolusi independen (bisa diperbesar tanpa kehilangan kualitas)
  • Ukuran file lebih kecil
  • Contoh format: SVG, AI, PDF

3. Sistem Koordinat 2D

Dua jenis koordinat:

  • Kartesian: Origin di tengah, Y positif ke atas
  • Canvas: Origin di kiri atas, Y positif ke bawah

üíª Contoh Kode

Gambar Piksel

const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');

// Gambar lingkaran
ctx.fillStyle = '#ff5733';
ctx.beginPath();
ctx.arc(200, 150, 60, 0, Math.PI * 2);
ctx.fill();

Gambar Vektor (Data)

const circle = {
  type: 'circle',

  radius: 60,

};

Konversi Koordinat

// Kartesian ke Canvas
function cartesianToCanvas(x, y) {
  const originX = canvasWidth / 2;
  const originY = canvasHeight / 2;
  return {
    x: originX + x,
    y: originY - y  // Y dibalik
  };
}
