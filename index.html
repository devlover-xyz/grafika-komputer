üé® Model Warna

RGB (Red, Green, Blue)

Model warna untuk layar digital. Setiap warna adalah kombinasi merah, hijau, dan biru.

üìù Source Code

function updateRGB() {
  const r = document.getElementById('rSlider').value;
  const g = document.getElementById('gSlider').value;
  const b = document.getElementById('bSlider').value;

  // Update display
  document.getElementById('rValue').textContent = r;
  document.getElementById('gValue').textContent = g;

  ctx.fillRect(200, 50, 100, 100);

  ctx.fillRect(350, 50, 100, 100);

  // Draw combined color
  ctx.fillStyle = `rgb(${r}, ${g}, ${b})`;
  ctx.fillRect(500, 50, 80, 100);
}

HSV (Hue, Saturation, Value)

Model warna intuitif untuk manusia. Hue = warna dasar, Saturation = intensitas, Value = kecerahan.

CMYK (Cyan, Magenta, Yellow, Black)

Model warna untuk printing. Subtractive color model - warna dihasilkan dari penyerapan cahaya.

üìù Source Code - Konversi Warna

// Konversi HSV ke RGB
function hsvToRgb(h, s, v) {
  s /= 100;
  v /= 100;

  const c = v * s;
  const x = c * (1 - Math.abs(((h / 60) % 2) - 1));
  const m = v - c;

  let r, g, b;
  if (h < 60) [r, g, b] = [c, x, 0];
  else if (h < 120) [r, g, b] = [x, c, 0];
  else if (h < 180) [r, g, b] = [0, c, x];
  else if (h < 240) [r, g, b] = [0, x, c];
  else if (h < 300) [r, g, b] = [x, 0, c];
  else [r, g, b] = [c, 0, x];

  return {
    r: Math.round((r + m) * 255),
    g: Math.round((g + m) * 255),
    b: Math.round((b + m) * 255)
  };
}

// Konversi CMYK ke RGB
function cmykToRgb(c, m, y, k) {
  c /= 100;
  m /= 100;
  y /= 100;
  k /= 100;

  const r = 255 * (1 - c) * (1 - k);
  const g = 255 * (1 - m) * (1 - k);
  const b = 255 * (1 - y) * (1 - k);

  return {
    r: Math.round(r),
    g: Math.round(g),
    b: Math.round(b)
  };
}

üîÑ Transformasi 2D

Konsep: Mengubah posisi, ukuran, atau orientasi objek menggunakan rumus matematis.

Jenis Transformasi

1. Translasi (Pergeseran)

x' = x + dx
y' = y + dy

2. Rotasi (Putaran)

x' = x √ó cos(Œ∏) - y √ó sin(Œ∏)
y' = x √ó sin(Œ∏) + y √ó cos(Œ∏)

3. Skala (Perbesaran/Perkecilan)

x' = x √ó sx
y' = y √ó sy

üìù Source Code

function updateTransform() {
  const tx = parseFloat(document.getElementById('txSlider').value);
  const ty = parseFloat(document.getElementById('tySlider').value);
  const rot = parseFloat(document.getElementById('rotSlider').value);
  const scale = parseFloat(document.getElementById('scaleSlider').value);

  ctx.clearRect(0, 0, 600, 400);

  // Original shape (gray)
  ctx.save();
  ctx.translate(300, 200);
  ctx.fillStyle = 'rgba(200, 200, 200, 0.5)';
  ctx.fillRect(-50, -30, 100, 60);
  ctx.restore();

  // Transformed shape (purple)
  ctx.save();
  ctx.translate(300 + tx, 200 + ty);  // Translasi
  ctx.rotate((rot * Math.PI) / 180);   // Rotasi
  ctx.scale(scale, scale);             // Skala
  ctx.fillStyle = '#667eea';
  ctx.fillRect(-50, -30, 100, 60);
  ctx.restore();
}

Transformasi Tambahan

4. Shear (Skewing) - Memiringkan

Shear Horizontal: x' = x + shx √ó y, y' = y
Shear Vertical: x' = x, y' = y + shy √ó x

5. Reflection (Pencerminan)

Terhadap X: x' = x, y' = -y
Terhadap Y: x' = -x, y' = y
Terhadap Origin: x' = -x, y' = -y
Diagonal (y=x): x' = y, y' = x

// Fungsi menggambar segitiga siku-siku
function drawTriangle(ctx, color, strokeColor) {
  // A: Kiri bawah (siku) = (-40, 40)
  // B: Kanan bawah = (40, 40)
  // C: Kiri atas = (-40, -40)

  ctx.beginPath();
  ctx.moveTo(-40, 40);    // Titik A
  ctx.lineTo(40, 40);     // Titik B
  ctx.lineTo(-40, -40);   // Titik C
  ctx.closePath();

  ctx.fillStyle = color;
  ctx.fill();
  ctx.strokeStyle = strokeColor;
  ctx.lineWidth = 2;
  ctx.stroke();

  // Label titik A, B, C
  ctx.fillStyle = strokeColor;
  ctx.font = 'bold 12px Arial';
  ctx.textAlign = 'center';
  ctx.textBaseline = 'middle';

  // Titik A (kiri bawah - siku)
  ctx.fillText('A', -40, 40 + 12);

  // Titik B (kanan bawah)
  ctx.fillText('B', 40, 40 + 12);

  // Titik C (kiri atas)
  ctx.fillText('C', -40 - 12, -40);
}

// Shear Transformation
function updateShear() {
  const shx = parseFloat(document.getElementById('shxSlider').value);
  const shy = parseFloat(document.getElementById('shySlider').value);

  ctx.clearRect(0, 0, 600, 400);

  // Original shape (gray) - segitiga
  ctx.save();
  ctx.translate(300, 200);
  drawTriangle(ctx, 'rgba(200, 200, 200, 0.5)', '#999');
  ctx.restore();

  // Sheared shape (purple) - segitiga
  ctx.save();
  ctx.translate(300, 200);
  ctx.transform(1, shy, shx, 1, 0, 0);  // Shear transformation
  drawTriangle(ctx, '#667eea', '#000');
  ctx.restore();
}

// Fungsi untuk menggambar segitiga siku-siku
function drawRightTriangle(ctx, fillColor, strokeColor) {
  // A: Kiri bawah (siku) = (-50, 50)
  // B: Kanan bawah = (50, 50)
  // C: Kiri atas = (-50, -50)

  ctx.beginPath();
  ctx.moveTo(-50, 50);    // Titik A
  ctx.lineTo(50, 50);     // Titik B
  ctx.lineTo(-50, -50);   // Titik C
  ctx.closePath();

  ctx.fillStyle = fillColor;
  ctx.fill();
  ctx.strokeStyle = strokeColor;
  ctx.lineWidth = 3;
  ctx.stroke();

  // Label titik A, B, C
  ctx.fillStyle = strokeColor;
  ctx.font = 'bold 14px Arial';
  ctx.textAlign = 'center';
  ctx.textBaseline = 'middle';

  // Titik A (kiri bawah - siku)
  ctx.fillText('A', -50, 50 + 15);

  // Titik B (kanan bawah)
  ctx.fillText('B', 50, 50 + 15);

  // Titik C (kiri atas)
  ctx.fillText('C', -50 - 15, -50);
}

// Reflection Transformation
function applyReflection(type) {
  ctx.clearRect(0, 0, 600, 400);

  // Original shape (gray) - segitiga siku-siku
  ctx.save();
  ctx.translate(300, 200);
  ctx.translate(70, -40);
  drawRightTriangle(ctx, 'rgba(200, 200, 200, 0.5)', '#999');
  ctx.restore();

  // Reflected shape (purple)
  ctx.save();
  ctx.translate(300, 200);

  switch(type) {
    case 'x':    // Mirror terhadap X-axis
      ctx.translate(70, -40);
      ctx.scale(1, -1);
      break;
    case 'y':    // Mirror terhadap Y-axis
      ctx.translate(70, -40);
      ctx.scale(-1, 1);
      break;
    case 'origin': // Mirror terhadap origin
      ctx.translate(70, -40);
      ctx.scale(-1, -1);
      break;
    case 'diagonal': // Mirror terhadap y=x
      ctx.translate(70, -40);
      ctx.transform(0, 1, 1, 0, 0, 0);
      break;
  }

  drawRightTriangle(ctx, '#667eea', '#000');
  ctx.restore();
}

üìê Matriks Transformasi

Konsep: Matriks 3√ó3 digunakan untuk melakukan transformasi 2D secara efisien. Koordinat homogen (x, y, 1) memungkinkan translasi dapat direpresentasikan dalam matriks.

1. Matriks Translasi (Geser)

Formula Matriks:

[1  0  dx]   [x]   [x + dx]
[0  1  dy] √ó [y] = [y + dy]
[0  0  1 ]   [1]   [  1   ]

Contoh Perhitungan Manual:

Geser titik (3, 2) sebesar dx=5, dy=4: 
[1  0  5]   [3]   [1√ó3 + 0√ó2 + 5√ó1]   [8]
[0  1  4] √ó [2] = [0√ó3 + 1√ó2 + 4√ó1] = [6]
[0  0  1]   [1]   [0√ó3 + 0√ó2 + 1√ó1]   [1]

Hasil: (3, 2) ‚Üí (8, 6)

2. Matriks Rotasi (Putar)

Formula Matriks:

[cos(Œ∏) -sin(Œ∏)  0]   [x]
[sin(Œ∏)  cos(Œ∏)  0] √ó [y]
[  0       0     1]   [1]

Contoh Perhitungan Manual:

Putar titik (4, 0) sebesar 90¬∞ (œÄ/2 radian): 
cos(90¬∞) = 0,  sin(90¬∞) = 1

[0  -1  0]   [4]   [0√ó4 + (-1)√ó0 + 0√ó1]   [0]
[1   0  0] √ó [0] = [1√ó4 + 0√ó0 + 0√ó1]    = [4]
[0   0  1]   [1]   [0√ó4 + 0√ó0 + 1√ó1]      [1]

Hasil: (4, 0) ‚Üí (0, 4)

3. Matriks Skala (Ubah Ukuran)

Formula Matriks:

[sx  0   0]   [x]   [x √ó sx]
[0   sy  0] √ó [y] = [y √ó sy]
[0   0   1]   [1]   [   1   ]

Contoh Perhitungan Manual:

Perbesar titik (2, 3) dengan sx=2, sy=1.5: 
[2   0    0]   [2]   [2√ó2 + 0√ó3 + 0√ó1]   [4.0]
[0   1.5  0] √ó [3] = [0√ó2 + 1.5√ó3 + 0√ó1] = [4.5]
[0   0    1]   [1]   [0√ó2 + 0√ó3 + 1√ó1]     [1]

Hasil: (2, 3) ‚Üí (4, 4.5)

4. Matriks Shear (Miring)

Formula Matriks Shear X:

[1   shx  0]   [x]   [x + shx√óy]
[0   1    0] √ó [y] = [    y    ]
[0   0    1]   [1]   [    1    ]

Contoh Perhitungan Manual:

Shear X pada titik (2, 4) dengan shx=0.5: 
[1   0.5  0]   [2]   [1√ó2 + 0.5√ó4 + 0√ó1]   [4.0]
[0   1    0] √ó [4] = [0√ó2 + 1√ó4 + 0√ó1]    = [4.0]
[0   0    1]   [1]   [0√ó2 + 0√ó4 + 1√ó1]      [1]

Hasil: (2, 4) ‚Üí (4, 4)

5. Matrix Reflection (Pencerminan)

üí° Konsep: Reflection (pencerminan) adalah transformasi yang memantulkan objek terhadap sumbu atau titik tertentu.
Jenis-jenis Reflection:
  • Reflection X-axis: Cermin terhadap sumbu X (horizontal) - y' = -y
  • Reflection Y-axis: Cermin terhadap sumbu Y (vertikal) - x' = -x
  • Reflection Origin: Cermin terhadap titik origin (0,0) - x' = -x, y' = -y
  • Reflection Diagonal: Cermin terhadap garis y=x - tukar x dan y

Contoh Perhitungan Manual:

Reflection terhadap sumbu X: 
Titik awal: (3, 2)

Matriks Reflection X:
[1   0  0]   [3]   [1√ó3 + 0√ó2 + 0√ó1]   [3]
[0  -1  0] √ó [2] = [0√ó3 + -1√ó2 + 0√ó1] = [-2]
[0   0  1]   [1]   [0√ó3 + 0√ó2 + 1√ó1]   [1]

Hasil: (3, 2) ‚Üí (3, -2)
Formula: x' = x, y' = -y
Reflection terhadap sumbu Y: 
Titik awal: (3, 2)

Matriks Reflection Y:
[-1  0  0]   [3]   [-1√ó3 + 0√ó2 + 0√ó1]   [-3]
[0   1  0] √ó [2] = [0√ó3 + 1√ó2 + 0√ó1] =   [2]
[0   0  1]   [1]   [0√ó3 + 0√ó2 + 1√ó1]    [1]

Hasil: (3, 2) ‚Üí (-3, 2)
Formula: x' = -x, y' = y
Reflection terhadap Origin: 
Titik awal: (3, 2)

Matriks Reflection Origin:
[-1  0  0]   [3]   [-1√ó3 + 0√ó2 + 0√ó1]   [-3]
[0  -1  0] √ó [2] = [0√ó3 + -1√ó2 + 0√ó1] = [-2]
[0   0  1]   [1]   [0√ó3 + 0√ó2 + 1√ó1]    [1]

Hasil: (3, 2) ‚Üí (-3, -2)
Formula: x' = -x, y' = -y
Reflection terhadap Diagonal (y=x): 
Titik awal: (3, 2)

Matriks Reflection Diagonal:
[0  1  0]   [3]   [0√ó3 + 1√ó2 + 0√ó1]   [2]
[1  0  0] √ó [2] = [1√ó3 + 0√ó2 + 0√ó1] = [3]
[0  0  1]   [1]   [0√ó3 + 0√ó2 + 1√ó1]   [1]

Hasil: (3, 2) ‚Üí (2, 3)
Formula: x' = y, y' = x (tukar koordinat)

const reflectionX = [1, 0, 0, -1, 0, 0];

// Reflection Y-axis (mirror vertical)
const reflectionY = [-1, 0, 0, 1, 0, 0];

  };
}

6. Transformasi Komposit (Gabungan)

üí° Konsep: Beberapa transformasi dapat digabungkan dengan perkalian matriks.
Urutan penting: M_final = M_translate √ó M_rotate √ó M_scale
Transformasi diterapkan dari kanan ke kiri (scale ‚Üí rotate ‚Üí translate)

Contoh Perhitungan Manual:

Transformasi komposit: Scale(2,2) ‚Üí Translate(3,1) 
Titik awal: (1, 2)

1. Scale (2, 2):
   [2  0  0]   [1]   [2]
   [0  2  0] √ó [2] = [4]
   [0  0  1]   [1]   [1]

2. Translate (3, 1):
   [1  0  3]   [2]   [5]
   [0  1  1] √ó [4] = [5]
   [0  0  1]   [1]   [1]

Hasil: (1, 2) ‚Üí (5, 5)

Atau dengan matriks komposit:
M = M_translate √ó M_scale
  = [1  0  3]   [2  0  0]   [2  0  3]
    [0  1  1] √ó [0  2  0] = [0  2  1]
    [0  0  1]   [0  0  1]   [0  0  1]

Verifikasi:
[2  0  3]   [1]   [5]
[0  2  1] √ó [2] = [5]
[0  0  1]   [1]   [1]  ‚úì Sama!

// Buat matriks translasi
function createTranslationMatrix(dx, dy) {
  return [1, 0, 0, 1, dx, dy];
  // Format: [a, b, c, d, e, f] = [a c e]
  //                               [b d f]
  //                               [0 0 1]
}

// Buat matriks rotasi
function createRotationMatrix(angle) {
  const rad = angle * Math.PI / 180;
  const cos = Math.cos(rad);
  const sin = Math.sin(rad);
  return [cos, sin, -sin, cos, 0, 0];
}

// Buat matriks scale
function createScaleMatrix(sx, sy) {
  return [sx, 0, 0, sy, 0, 0];
}

// Kalikan dua matriks
function multiplyMatrices(m1, m2) {
  const [a1, b1, c1, d1, e1, f1] = m1;
  const [a2, b2, c2, d2, e2, f2] = m2;

  return [
    a1*a2 + c1*b2,           // a
    b1*a2 + d1*b2,           // b
    a1*c2 + c1*d2,           // c
    b1*c2 + d1*d2,           // d
    a1*e2 + c1*f2 + e1,      // e
    b1*e2 + d1*f2 + f1       // f
  ];
}

// Aplikasikan matriks ke titik
function applyMatrix(point, matrix) {
  const [a, b, c, d, e, f] = matrix;
  return {
    x: a * point.x + c * point.y + e,
    y: b * point.x + d * point.y + f
  };
}
